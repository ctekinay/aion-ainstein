# ArchiMate 3.2 — Visual Shaping Reference

Standard element shapes and layer colors as defined by the ArchiMate 3.2 specification.
These are rendering conventions — they are **not expressed in Open Exchange XML**.
Any viewer or renderer (browser-based, tooling, or otherwise) should follow these conventions.

---

## Element Visual Shapes

| Shape type | Element examples |
|---|---|
| Rectangle | Structural: Actor, Component, Node, Object, DataObject |
| Rounded | Behavioral: Process, Function, Service, Event, Interaction |
| Ellipse | Value |
| Motivation | Goal, Driver, Requirement, Constraint, Principle, Assessment |
